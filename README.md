# Bangalore House Price Prediction

This project predicts Bangalore house prices using Machine Learning techniques and data analysis.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing
- Feature engineering
- Outlier removal
- Data visualization
- House price prediction using regression models

## Workflow
1. Load and clean housing dataset
2. Perform feature engineering and preprocessing
3. Train regression model
4. Predict house prices based on input features
